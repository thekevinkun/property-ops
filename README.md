## 1. What Is This Project

This is a **pilot-ready multi-role SaaS web application** built as a test task for Adriano — a potential client looking for a senior full-stack developer to build an MVP for professional short-stay, business-apartment, and serviced-apartment operators.

### What It Demonstrates

- Multi-role authentication and RBAC with three distinct roles: **Host**, **Operator**, and **Admin**
- A property entity with real relational data
- Tasks with enforced status transition logic (state machine)
- Evidence/photo uploads via Supabase Storage
- An append-only audit log that records every meaningful action automatically
- Clean, maintainable architecture that Adriano can read, clone, and evaluate
